# Servidor Escuelab

Es una adptación y mejora del *Servidor Escuela PeruEduca* del Ministerio de Educación de Perú.

Contiene los siguientes componentes:

- Ubuntu LTS
- KhanAcademy
- Emby Server
- Moodle
- NextCloud
- Kiwix
  - Wikipedia
  - Phet
  - Wikcionario
- Wordpress Network
- Drupal
- OpenLDAP
- LetsEncrypt *
- Squid Proxy
